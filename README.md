#### Repository Contents



This repository includes the following shell scripts developed as part of the assignment:



* script1\_system\_identity.sh



Displays basic system information:



Hostname

Current user

Operating system

Kernel version

* script2\_package\_inspector.sh

Checks whether a specified software package is installed

Displays relevant package details

* script3\_disk\_auditor.sh

Shows disk usage statistics

Checks file permissions in selected directories

* script4\_log\_analyzer.sh

Reads system log files

Extracts important data such as:

Error messages

Warnings

* script\_5\_manifesto.sh

Generates a text output highlighting the importance of open-source software



#### Requirements



To run the scripts, ensure the following:



Linux OS (Ubuntu / Debian / Fedora)

OR Windows with WSL (Windows Subsystem for Linux)

Bash shell

Basic terminal access
